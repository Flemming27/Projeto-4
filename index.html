O Sloga do GitHub é "Social Code Host" (hospedeiro social de códigos, em tradução livre). A base do site é justamente essa: armazenar códigos de programação, produzidos por desenvolvedores do mundo todo, e compartilhá-los como se fosse uma rede social. Dessa forma é possível que quaisquer usuários cadastrados na plataforma divulguem seus trabalhos e que outros membros da comunidade façam contribuições. Meu GitHub: https://github.com/Flemming27
